# Gerenciador de suprimentos
Controle de estoque e de retiradas de suprimentos de impressora

## Funcionalidades

-> Cadastro de suprimentos  
-> Controle de retiradas de material  
-> Mapeamento das impressoras  
-> Geração de relatórios para excel  
-> Integração com o PIPEFY  

## Requisitos

-> Acesso à internet

## Uso

Link para acesso web: [https:\\ticambai.netlify.app](https://ticambai.netlify.app/)
